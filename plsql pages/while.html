PL SQL WHILE LOOP:

The pl sql while loop repeatedly executes a block of statements until a particular condition is true. It first check the condition and executes a block of statements if condition is true.


PL SQL WHILE LOOP syntax:

Syntax :

WHILE condition
LOOP
  //block of statements;
END LOOP;
